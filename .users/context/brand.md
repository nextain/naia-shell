# Brand Context

> **SoT**: `.agents/context/brand.yaml`

## Origin

Both **Naia** and **Nextain** are born from the same words: **N**ext **AI** Networks.

- **Nextain**: The company
- **Naia**: The AI character who lives inside the OS. The shell is the OS, Naia is the soul
- **Slogan**: Next AI Networks
- **Brand Principle**: Use "Naia" without OS suffix — like Windows, macOS, Bazzite
- **Nextain Meaning**: Not just a homage to NHN (Next Human Networks). Nextain means the next-generation internet where humans and AI, and AI with AI, build together
- **Initial Committer**: [Luke](https://github.com/luke-n-alpha)
- **Korean Persona**: Official name is '나이아(Naia)', but the avatar greets with '나야(Na-ya)!' as a pun

## History

- Original name: **NaN** (Next AI Networks)
- Renamed reason: existing OS brand conflict
- Current name: **Naia**

## Character Design

- **Material**: Translucent frosted glass (glassmorphism), smooth glossy finish
- **Internal**: Glowing Flow Cyan (#06B6D4) digital circuit patterns and micro-node networks
- **Eyes**: Large, solid black glossy orbs
- **Mouth**: Minimalist 'w' shaped cat mouth
- **Ears**: Cute cat-like ears
- **Tail**: Thin, translucent cat tail (back view)
- **Hair** (optional): Bob-style, gradient from Nextain Blue (#2563EB) to Evolution Green (#10B981), holographic sheen
- **Logo**: Glowing cyan equilateral triangle with nodes at vertices, floats above head as halo
- **Chest Emblem**: Triangle logo + 'Naia' text etched in glowing cyan
- **Variants**: With hair (bob-style gradient) / Without hair (clean android style)

## Color System

| Name | HEX | Role |
|---|---|---|
| Nextain Blue | `#2563EB` | Trust, network base, gradient start |
| Flow Cyan | `#06B6D4` | Data flow, AI connection, hover/progress |
| Evolution Green | `#10B981` | Growth, ecosystem, gradient end |
| Terminal Black | `#171717` | Main text, logo font |
| Cloud White | `#F8FAFC` | Light mode background |
| Deep Node | `#0F172A` | Dark mode background |

**Gradient direction**: Bottom-left (#2563EB) to top-right (#10B981), diagonal linear
