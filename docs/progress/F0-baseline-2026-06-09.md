## Golden reference 거동 (코드 도출)

| 입력 상태 | 거동 (출력/상태) | disposition |
|---|---|---|
| 설정 없음(first run) | `AdkSetupScreen` 표시 (crash X) | **contain**(정직 안내) |
| 설정 있음·키 없음 | main 부팅, auth_update 스킵, workspace ready | 정상(외부키X 부팅) |
| 설정 있음·키 있음 | main 부팅 + auth_update 전송 | 정상(+외부) |
| workspace root ≠ dir / inaccessible | `workspace_set_root` → `Err` | **block/fail-closed**(무결성) |

## 상태 전이 (hidden state)

## 오류 분류 (F0 taxonomy 적용)
- workspace root invalid/inaccessible = **integrity-critical → block**(fail-closed).
- 설정 부재 = **optional → contain**(AdkSetupScreen 안내, 부팅 차단 X).
- naiaKey 부재 = degradation 아님(정상 무키 모드).
